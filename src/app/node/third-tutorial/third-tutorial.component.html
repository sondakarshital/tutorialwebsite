Let's start with first nodeJs Example.

app/firstNodeJsExample.js

1.First line of code imports http module by using require keyword and provides http instance.

2.Line 3 creates node server using http module and returns hello nodejs world! to browser.

Running node example

1.Go to directory containing firstNodeJsExample.js file and issue node firstNodeJsExample.js command

Observations

1.Once node command is issued it prints Server running at http://127.0.0.1:8080/ in command line.

2.Now the node sever is running on 8080 port.

2.Open your favourite browser and hit http://127.0.0.1:8080/ and you will see hello nodejs world! in a browser.

That's it.We have created our first nodeJs example.
