Setting up nodeJs environment and code editors

Setting up nodeJs environment is pretty easy.

1. Go to nodejs official page https://nodejs.org/en/download/current/ and download node current version.

2. Double click and install downloaded file.

3. Once installed successfully go to command line and issue npm -v and you should see installed version .

That's it,we have set up node in our system.We will see first nodeJs example in next tutorial.
